# K-means-clustering_on-_credit_card_data
Credit Card Customer Segmentation using K-Means
This project analyzes Credit Card Customer Data to segment users based on their Annual Income and Spending Score using K-Means clustering. The goal is to identify distinct customer groups for better marketing strategies.

Key Features:
Dataset: Credit Card Customer Data
Clustering Algorithm: K-Means (n_clusters=4)
Visualization: Scatter plots with distinct colors for clusters
Centroid Plotting: Highlighting cluster centers
Insights: Helps in customer segmentation and targeted marketing

Usage:
Load Credit_Card_Customer_Data.csv

Train the K-Means model on Annual Income & Spending Score

Visualize segmented customer groups

This project enables data-driven customer analysis for improved business strategies.
